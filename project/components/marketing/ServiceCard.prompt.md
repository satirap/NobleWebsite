Service offering card with a gradient icon tile, title, description, and animated arrow CTA. Built for The Noble's "บริการของเรา" service grid.

```jsx
<ServiceCard
  icon={<i data-lucide="server"></i>}
  title="Computer & Server"
  description="วางระบบคอมพิวเตอร์ เซิร์ฟเวอร์ NAS พร้อมใช้งาน"
  href="/services/computer-server/" />
```

Composes `Card`. Props: `icon`, `image` (photo URL for a 16:10 header), `title`, `description`, `href`, `cta`.
