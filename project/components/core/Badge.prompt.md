Small tonal pill for statuses, categories, and counts. Soft fill by default; solid for stronger emphasis.

```jsx
<Badge tone="brand">System Integration</Badge>
<Badge tone="success" dot>Online</Badge>
<Badge tone="accent" variant="solid" size="sm">ใหม่</Badge>
```

Tones: neutral, brand, accent, success, warning, danger. `variant="soft"|"solid"`, `size="sm"|"md"`, `dot` for a status indicator.
